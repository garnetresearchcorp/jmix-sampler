Dialog facets - это невизуальные компоненты, позволяющие декларативно настроить стандартные диалоговые окна в XML дескрипторе экрана. Диалоговые окна могут быть настроены с помощью следующих невизуальных компонентов:

  • MessageDialogFacet - соответствует вызову Dialogs#createMessageDialog;
  • OptionDialogFacet - соответствует вызову Dialogs#createOptionDialog;
  • InputDialogFacet - соответствует вызову Dialogs#createInputDialog.

Диалоговые окна могут быть открыты с помощью с помощью вызова соответсвующего метода: show() или с помощью подписки на действие или кнопку по идентификатору:

  • onButton - определяет идентификатор кнопки, которая вызовет показ диалогового окна;
  • onAction - определяет идентификатор действия, которое вызовет показ диалогового окна.
