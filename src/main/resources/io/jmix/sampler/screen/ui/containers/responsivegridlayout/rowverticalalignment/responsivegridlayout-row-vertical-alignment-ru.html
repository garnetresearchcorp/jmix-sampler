Вы можете выровнять содержимое строки вертикально используя атрибут alignItems в определенном breakpoint.

Возможны следующие значения:

  • BASELINE - каждая колонка внутри строки выравнивается по их базовой линии.
  • CENTER - каждая колонка внутри строки выравнивается по центру.
  • END - каждая колонка внутри строки выравнивается по нижнему краю.
  • START - каждая колонка внутри строки выравнивается по верхнему краю.
  • STRETCH - каждая колонка внутри строки растягивается на всю высоту строки.
