setStyleProvider()позволяет задать стиль отображения ячеек таблицы.

Для GroupTable данный метод будет принимать GroupTable.GroupStyleProvider, который расширяет Table.StyleProvider.

GroupStyleProvider имеет специальный метод для стилизации групировочных строк с GroupInfo в качестве принимаемого параметра. Он будет вызываться для каждой группировочной строки в GroupTable.
