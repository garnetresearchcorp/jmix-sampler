Вы можете выровнять отдельные col с помощью атрибута alignSelf в определенном breakpoint.

Возможны следующие значения:

  • AUTO - col берет значение из родительского свойства alignItems.
  • BASELINE - col выравнивается по его базовой линии.
  • CENTER - col выравнивается по центру.
  • END - col выравнивается по нижнему краю.
  • START - col выравнивается по верхнему краю.
  • STRETCH - col растягивается на всю высоту строки.
