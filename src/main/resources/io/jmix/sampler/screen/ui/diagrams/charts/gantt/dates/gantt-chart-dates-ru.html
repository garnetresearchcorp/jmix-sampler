Данная версия Gantt-диаграммы показывает пример использования абсолютных дат для определения начала и конца сегментов, вне зависимости от того, где заканчивается другой сегмент в этой же категории.

Для определения, какие поля сущности сегмента содержат значения для даты начала и окончания, используются атрибуты startDateField и endDateField соответственно.

Обратите внимание, что это демо также использует атрибут brightnessStep. Это позволяет постепенно увеличивать яркость каждого последующего сегмента.
