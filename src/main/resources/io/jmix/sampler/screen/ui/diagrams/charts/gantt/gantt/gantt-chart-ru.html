Gantt-диаграмма - представляет собой модификацию гистограммы и отображает несколько сегментов, не обязательно связанных друг с другом. Она может быть использована, чтобы отметить промежутки времени, даты или значений.

Наиболее часто Gantt-диаграмма используется для иллюстрации плана, графика работ по какому-либо проекту. Является одним из методов планирования проектов.
