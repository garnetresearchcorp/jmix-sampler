На карте могут быть размещены точки. Для отображения точки, по умолчанию используется стандартная синяя иконка, которую можно заменить на иконку следующих типов: FontPointIcon, ImagePointIcon, DivPointIcon.

В данном примере новая точка добавляется при щелчке на карте. Щелчок на точке открывает всплывающее окно с информацией.
