ResponsiveGridLayout поддерживает два режима:

  • FLUID - контейнер занимает всю ширину breakpoint, т.е. имеет ширину 100%. Является значением по умолчанию.
  • FIXED - как только экран достигает определенного breakpoint, ширина ResponsiveGridLayout устанавливается в следующие значения:
    • xl - 1140px
    • lg - 960px
    • md - 720px
    • sm - 540px
