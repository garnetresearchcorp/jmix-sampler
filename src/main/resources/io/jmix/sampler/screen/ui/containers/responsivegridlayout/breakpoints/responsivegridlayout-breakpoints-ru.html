Всё горизонтальное пространство ResponsiveGridLayout поделено на 12 частей одинаковой ширины. Каждая колонка может занимать одну или более частей, и Вы можете настроить это отдельно для каждого размера экрана (или breakpoints, в терминологии Bootstrap). Breakpoints определены атрибутами xs, sm, md, lg, xl и значение атрибута задает ширину колонки в 1/12 частях для заданной ширины экрана.

Ниже представлено соотношение между breakpoints и размерами экранов:

  • xs - для очень маленьких экранов, с шириной меньше 576px. Является значением по умолчанию.
  • sm - для маленьких экранов, с шириной между 576px и 768px.
  • md - для средних экранов, с шириной между 768px и 992px.
  • lg - для больших экранов, с шириной между 992px и 1200px.
  • xl - для очень больших экранов, с шириной 1200px и больше.
