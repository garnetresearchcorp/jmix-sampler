По умолчанию пользователь не может ввести значение вручную. Если вы установите для атрибута fieldEditable значение true, вы включите ручной ввод. Имейте в виду, что введенное значение не устанавливается в модель данных. Для обработки пользовательского ввода используйте обработчик FieldValueChangeEvent.
