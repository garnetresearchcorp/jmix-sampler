Атрибут justifyContent может быть исполдьзовать для задания горизонтального выравнивания в определенном breakpoint.

Возможны следующие значения:

  • AROUND - слева, справа и между холодами есть отступы.
  • BETWEEN - между колонкамиесть отступы.
  • CENTER - колонки внутри строки выравнивается по центру.
  • END - колонки внутри строки выравнивается по правому краю.
  • START - колонки внутри строки выравнивается по левому краю.
