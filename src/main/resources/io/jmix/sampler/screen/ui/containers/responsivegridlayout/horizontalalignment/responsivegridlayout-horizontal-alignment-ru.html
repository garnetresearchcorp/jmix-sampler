Атрибут justifyContent может быть использован для задания горизонтального выравнивания в определенном breakpoint.

Возможны следующие значения:

  • AROUND - слева, справа и между колонками есть отступы.
  • BETWEEN - между колонками есть отступы.
  • CENTER - колонки внутри строки выравниваются по центру.
  • END - колонки внутри строки выравниваются по правому краю.
  • START - колонки внутри строки выравниваются по левому краю.
