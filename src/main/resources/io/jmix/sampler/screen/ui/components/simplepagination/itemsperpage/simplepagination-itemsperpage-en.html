SimplePagination can manage the maximum size of the page. This functionality is presented by the following properties:

  • itemsPerPageVisible - defines whether the dropdown list of maximum page size options should be visible.
  • itemsPerPageOptions - sets custom maximum page size options that should be used in the dropdown list. Values should be separated by the comma, e.g. "10, 25, 50".
  • itemsPerPageDefaultValue - sets the default value from available options that should be used by default. It can be used even if the itemsPerPageVisible attribute is set to "false".
  • itemsPerPageUnlimitedOptionVisible - defines should be option that is shown as empty item in ComboBox visible. If this option is selected the maximum page size will be provided by the entityPageSize application property.
