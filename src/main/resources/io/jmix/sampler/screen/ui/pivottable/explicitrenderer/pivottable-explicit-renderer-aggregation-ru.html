Атрибут renderer представляет собор выбор из списка предустановленных рендереров, служащих для отображения данных.

Элемент aggregation отвечает за настройки для функции, которая будет использоваться для агрегирования значений в каждой ячейке.

Атрибуты aggregation:

  • mode - выбор из списка предопределенных функций.
  • caption - локализуемое значение, которое будет отображаться в UI.
  • custom - если true, то значение mode игнорируется, и будет использоваться JavaScript код, заданный во вложенном элементе function.

Элементы aggregation:

  • function - JavaScript код, используемый в качестве функции агрегирования.
  • property - список атрибутов, которые будут использоваться в качестве входных параметров функции агрегирования. Значением может быть либо ключ из properties, либо имя сгенерированного атрибута.
