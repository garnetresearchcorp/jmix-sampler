SimplePagination может отображать общее количество записей, возращаемое текущим запросом в контейнере данных без извлечения самих записей. Когда пользователь нажимает на знак "?", компонент выполняет запрос в базу данных с условиями, которые используются в текущем запросе, но с агрегатной функцией COUNT(*) вместо результатов. Полученное число отображается вместо знака "?".
