Компонент ValuesPicker работает со списком значений любого типа. Он состоит из текстового поля и набора кнопок, определяемых действиями.

ValuesPicker удобно использовать, если вам нужно редактировать список значений с помощью действий, а не в текстовом поле.

Вы можете определить пользовательские и предопределенные действия для ValuesPicker, отображаемые в виде кнопок справа.

Framework предоставляет предопределенные действия value_clear и values_select.

Действие values_select устанавливает значение для ValuesPicker с помощью экрана выбора значения. Экран выбора значения генерирует определенный компонент для выбора или создания значения в зависимости от типа переданного значения.
