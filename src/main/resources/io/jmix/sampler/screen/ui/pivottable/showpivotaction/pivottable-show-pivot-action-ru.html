ShowPivotAction - действие, которое позоволяет показать данные из таблицы в отдельном экране с компонентом PivotTable. Данное действие можно использовать как из XML-дескриатора экрана, так и из его контроллера.

PivotScreenBuilder - это Spring Prototype компонент. Он предоставляет API для настройки PivotTable в открывающемся экране.
